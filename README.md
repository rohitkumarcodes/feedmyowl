# FeedMyOwl

Reading-first RSS/Atom reader.

## Local Development

See `/Users/rohitkumar/master-directory/code/feedmyowl/RUNBOOK.md` for the full runbook.

Quick start:

```bash
pnpm install
pnpm dev:web
pnpm dev:blog
```

Useful checks:

```bash
pnpm format:check
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
