# FeedMyOwl

Reading-first RSS/Atom reader.

## Local Development

See `RUNBOOK.md` for the full runbook.

Quick start:

```bash
pnpm install
pnpm dev:web
pnpm dev:blog
```

Useful checks:

```bash
pnpm ui:check:web
pnpm check:architecture
pnpm format:check
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

UI routine:

- Follow the UI checklist in `RUNBOOK.md` section `2.1 UI change check routine` for every UI task.

Agent workflow:

- Start with `AGENT_TASK_GUIDE.md` for copy-paste task prompts.
