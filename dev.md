# FeedMyOwl Developer Notes

`AGENTS.md` is the source of truth for architecture, commands, module
boundaries, database rules, and coding conventions. Start there for every coding
task.

This file only exists as a friendly pointer for humans and coding agents that
open `dev.md` by habit.

Useful companion docs:

- `AGENT_TASK_GUIDE.md` — copy-paste prompts for common coding-agent tasks.
- `RUNBOOK.md` — local development, UI verification, and incident checks.
- `FEEDMYOWL_PROJECT_CONTEXT.md` — product scope and current behavior.
- `DESIGN.md` and `UX.md` — interface and interaction expectations.

Important reminder: the web app uses plain CSS Modules, not Tailwind.
